Sobre a ~SALTLIFE~

CRIADOR

Meu nome é Bruno Leonardo, um aluno da BandTec cursando CCO. Meu gosto por tênis veio a um bom tempo e só vem aumentando...
Sempre senti uma falta de uma cominudade ou algo dedicado somente a os amantes de sneakers e por isso criei a SALTLIFE,
onde as pessoas podem a cada dia mais se conversarem

Nossos Objetivos

Criar uma Comunidade

Sinto falta de uma comunidade especifica para os amantes de calçados interagirem.

Juntar "TENNIS LOVERS"

Atrair e acomchegar novos usuarios para a nossa plataforma.

Diminuir informações falsas

Com essas noticias falsas, criamos os moderadores para diminuir esse fluxo melhorando a comunidade.

Aumentar a comunidade
